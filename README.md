# agri-links 🌱

農業資材メーカーの公式サイト・電子カタログ・PDFリンクを整理した一覧サイトです。

現場で

* 「あのメーカーのカタログどこだっけ？」
* 「PDF探しづらい…」
* 「メーカーサイトにたどり着けない」

となることが多かったため、
自分用に整理し始めたものを公開しています。

---

## Features

* 農業資材メーカー一覧
* 公式サイトへのリンク
* 電子カタログへのリンク
* PDFカタログへのリンク
* スマホ対応
* シンプルなHTML構成

---

## Purpose

このサイトは、

「探す時間を減らす」

ことを目的に作成しています。

農業系メーカーの情報は、
サイト構成やカタログ配置がバラバラなことも多いため、
よく使う情報へすぐアクセスできるよう整理しています。

---

## Notes

* 掲載しているリンク・カタログ等の権利は各メーカー様に帰属します。
* リンク先は各メーカー公式サイトです。
* メーカーによってはカタログが公開されていない場合があります。
* リンク切れ等がある場合はご了承ください。
